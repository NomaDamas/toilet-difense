## ▶️ 실행 (10초)

브라우저로 [https://nomadamas.github.io/toilet-difense](https://nomadamas.github.io/toilet-difense/) 열면 끝.

**로컬에서 돌리려면**:

```bash
python3 -m http.server 9999
# 그 다음 http://127.0.0.1:9999/
```

또는:

```bash
npx --yes serve -l 9999 .
```

ES modules라 `file://`로 열면 안 됩니다. HTTP로 서빙해야 합니다. **빌드 없음. 의존성 0. 백엔드 없음.**

## 🎮 조작법

마우스로 다 됩니다. 하지만 키보드 알면 더 잘합니다.

### 기본

| 키 | 동작 |
|---|---|
| `1` `2` `3` `4` | 타워 선택 (뚫어뻥 / 석션 / 세정제 / 배관공) |
| 슬롯 클릭 | 설치 |
| 타워 클릭 | 업그레이드 · 매각 |
| `T` | 타겟 모드 변경 (선두 / **위협** / 최강 / 후미 / 근접) |
| `Q` | 영웅 능력 발동 |
| `Space` | 일시정지 |

### 능동 도구 (타워 디펜스 + α)

타워만 짓는 게 아닙니다. 위기엔 직접 개입하세요:

| 키 | 도구 | 비용 | 효과 |
|---|---|---|---|
| `E` | 🚿 응급 분사 | **무료** | 클릭 지점 90px 적 밀어내기 + 데미지 |
| `R` | 🛑 임시 차단막 | ₩30,000 | 5초간 적 차단 (스턴) |
| `F` | 📞 김반장 출장 | ₩150,000 | 보스 −320 + 전체 3초 스턴 (면역 무시) |

### 편의

| 키 | 동작 |
|---|---|
| `+` `-` | 게임 속도 1× / 2× / 3× |
| `M` | 사운드 토글 |
| `O` | 설정 (볼륨 슬라이더 · 색맹 모드) |
| `U` | 5초 내 직전 판매 취소 |
| `H` / `?` | 단축키 도움말 |
| `Esc` / 우클릭 | 메뉴 닫기 |

| **`↑ ↑ ↓ ↓ ← → ← → B A`** | 🎮 코나미. ₩999,999 + 30초 무적 + 라이프 +10 |

## 🎯 적과 영웅과 보스

### 적

- 🧻 **물티슈** — 화학 저항. 뚫어뻥/석션
- 📜 **휴지** — 양반. 무리지어 옴
- 👁 **콘택트렌즈** — 35% 회피. 렌즈 용해제로 즉사
- 💊 **인공눈물 캡** — 죽으면 2개로 분열
- 🟨 **키친타월** — 엘리트. **절대 금물.** 공업용 석션이 ×2.4
- 👹 **정체불명의 물질** — 보스

### 영웅 3명

- 💰 **철사블** — 시작 자금 +30%. "₩80,000으로 화면 일소"
- 📋 **제프리** — 타워 8% 할인. "Rules Poster 무료, 4초 혼란"
- 🩺 **민성진** — 적 HP 상시 표시 · 모든 타워 +10%. "6초간 +50% DMG + 보스 면역 무효"

### 보스가 진짜 보스인 이유

- HP **6,000** · 속도 14 (느림) · 갑옷 5
- **넉백 92% 저항** (진공 뚫어뻥에 안 밀림) · 스턴 85% 저항 · 슬로우 70% 저항
- **적응형 면역**: 매 5.5초마다 4종 타워 면역을 자동 로테이션. 같은 타워만 쌓으면 안 통함
- **산성 분사**: 매 10초 가장 가까운 타워 3초 비활성화
- **HP < 50%**: 격분. 속도 +35%. 미세 잔해 2마리 5초마다 spawn
- **HP < 15%**: 광폭. 속도 ×2.2. 모든 저항 해제

→ 단일 타워로 잡기 불가능. **4종 타워 균형 + `F` 김반장 출장 타이밍**이 정답.
